# Systemfehler – Plattformübersicht für Fachkräfte
**Platform Overview for Domain Experts**

*Diese Seite richtet sich an Sozialarbeiter:innen, Rechtsberater:innen, NGO-Mitarbeiter:innen
und andere Fachkräfte, die mit Systemfehler zusammenarbeiten möchten.*

*This page is written for social workers, legal advisors, NGO staff, and other
domain professionals who want to work with Systemfehler.*

---

## Was ist Systemfehler? / What is Systemfehler?

**DE:** Systemfehler ist ein erster Orientierungspunkt und ein transparentes
Informationswerkzeug für Menschen in schwierigen bürokratischen Situationen –
zum Beispiel bei Fragen zu Bürgergeld, Jobcenter-Bescheiden, Sozialleistungen,
Sanktionen oder Anträgen. Nutzer:innen stellen eine Frage in eigenen Worten,
und das System liefert eine verständliche, quellenbasierte Orientierung: Was
könnte relevant sein, woher kommt die Information, und wo finden sie passende
weitere Hilfe?

**EN:** Systemfehler is a first point of orientation and a transparent
information tool for people navigating difficult bureaucratic situations, such
as questions about Buergergeld, Jobcenter decisions, social benefits,
sanctions, or applications. Users ask a question in plain language, and the
system returns clear, source-based orientation: what may be relevant, where the
information comes from, and where they can find appropriate further help.

---

## Wie funktioniert das System? / How does it work?

Das System arbeitet in drei Schritten:

**1. Frage verstehen**
Das System liest die Frage und versucht zu verstehen, in welcher Lebenslage
sich die Person befindet – zum Beispiel: gerade arbeitslos geworden, Sanktion
erhalten, Widerspruch planen, Antrag stellen wollen. Diese „Lebenslage" hilft
dabei, passende Informationen zu finden.

**2. Passende Informationen suchen**
Das System durchsucht eine kuratierte Datenbank mit Einträgen zu sozialen
Leistungen, Rechten, Pflichten und Verfahren. Dabei werden nur Einträge
ausgewählt, die thematisch zur Frage passen – zum Beispiel Einträge zu
Sanktionen, wenn jemand nach einer Kürzung fragt.

**3. Antwort formulieren**
Die gefundenen Informationen werden zu einer zusammenhängenden Antwort
zusammengefasst. Die Antwort kann in verschiedenen Sprachmodi angezeigt
werden:
- **Standard** – klare, direkte Sprache
- **Einfach** – vereinfachte Sprache für Menschen mit eingeschränkten
  Deutschkenntnissen oder Leseproblemen
- **Leicht** – Leichte Sprache nach offiziellen Regeln

## Was kann das System? / What can the system do?

| Situation | Beispiel-Frage |
|-----------|----------------|
| Bürgergeld-Fragen | „Wie hoch ist mein Bürgergeld?" |
| Sanktionen / Minderungen | „Ich habe eine Sanktion bekommen – was kann ich tun?" |
| Widerspruch einlegen | „Wie lege ich Widerspruch gegen einen Bescheid ein?" |
| Anträge stellen | „Wie beantrage ich Bürgergeld?" |
| Rechte kennen | „Darf das Jobcenter das von mir verlangen?" |
| Weitere Hilfe finden | „Wo bekomme ich kostenlose Beratung?" |

Das System gibt **Orientierung und Information**. Es soll helfen, schneller zur
richtigen Quelle, zum passenden Werkzeug oder zur passenden Beratungsstelle zu
finden. Es ersetzt keine Rechtsberatung und trifft keine rechtlich
verbindlichen Aussagen.

---

## Was kann das System **nicht**? / What the system **cannot** do

- Es kann keine Rechtsberatung ersetzen.
- Es kennt keine individuellen Falldetails (Bescheide, Akten, persönliche
  Situation).
- Es kann keine Anträge ausfüllen oder einreichen.
- Es kann nicht garantieren, dass alle Informationen immer tagesaktuell sind.
- Es gibt keine Empfehlungen für bestimmte Anwälte, Beratungsstellen oder
  Organisationen.

> **Wichtig:** Wenn eine Antwort im System fehlt oder falsch wirkt, bitte
> direkt Rückmeldung geben (siehe Abschnitt „Feedback" unten).

---

## Woher kommen die Informationen? / Where do the answers come from?

Die Informationsdatenbank enthält Einträge aus geprüften Quellen:
- Offizielle Behördeninformationen (z. B. Bundesagentur für Arbeit)
- Rechtsberatungsangebote anerkannter Organisationen
- Fachlich geprüfte Texte von gemeinnützigen Trägern

Alle Einträge folgen einem einheitlichen Schema mit Angaben zu Quelle,
Gültigkeitsbereich und Datum der letzten Prüfung. Das System zeigt nur
Informationen an, für die eine passende Quellenangabe vorliegt.

---

## Wie werden Fehler korrigiert? / How are errors corrected?

Das System enthält einen **redaktionellen Überprüfungsmechanismus**:

1. Wenn eine Frage unklar ist oder die Lebenslage nicht eindeutig erkannt
   wird, markiert das System die Antwort intern als „zur Überprüfung
   vorgemerkt".
2. Redakteur:innen können im Admin-Bereich überprüfen, welche Anfragen
   solche Markierungen erhalten haben.
3. Für wiederkehrende Problemfragen kann eine **Korrektur-Regel** angelegt
   werden: Das System nutzt dann bei ähnlichen Anfragen automatisch eine
   verbesserte Zuordnung.

Dies bedeutet: **Jedes gemeldete Problem kann dauerhaft behoben werden** –
nicht nur für eine Person, sondern für alle, die dieselbe Frage stellen.

## Wie können Fachkräfte helfen? / How can domain experts contribute?

Als Fachkraft mit Praxiswissen können Sie auf verschiedene Weisen beitragen:

### 1. Antworten testen
Stellen Sie typische Fragen aus Ihrer Beratungspraxis und prüfen Sie, ob die
Antworten inhaltlich korrekt und vollständig sind. Eine Anleitung dazu finden
Sie in der [Testanleitung für Expert:innen](expert-pilot-testing-guide.md).

### 2. Fehler melden
Wenn eine Antwort fehlt, unvollständig oder falsch ist: Notieren Sie die
genaue Frage und was die Antwort enthalten sollte. Melden Sie dies über das
Feedbackformular oder direkt an das Systemfehler-Team.

### 3. Inhalte vorschlagen
Wenn Sie wissen, dass ein wichtiges Thema fehlt (z. B. eine bestimmte
Leistungsart oder ein häufiges Verfahrensproblem), können Sie neue
Informationseinträge vorschlagen. Das Team prüft und integriert diese.

### 4. Sprachliche Rückmeldung geben
Wenn eine Antwort sprachlich schwierig oder missverständlich ist – besonders
im Modus „Einfach" oder „Leicht" – ist Ihre Einschätzung aus der Praxis
besonders wertvoll.

## Für wen ist das System gedacht? / Who is the target audience?

Das System richtet sich primär an:
- Menschen in Bürgergeld-Bezug oder auf dem Weg dahin
- Menschen, die mit Sanktionen oder Bescheiden konfrontiert sind
- Menschen, die Orientierung im deutschen Sozialsystem suchen
- Menschen mit eingeschränkten Deutschkenntnissen (dank einfacher
  Sprachmodi)

Es ist **kein Ersatz für persönliche Beratung**, sondern ein erster
Orientierungspunkt – vergleichbar mit einem gut informierten Freund, der
die wichtigsten Fakten kennt und erklärt, welche nächsten Schritte sinnvoll
sein könnten.

## Häufige Fragen / FAQ

**Ist die Nutzung kostenlos?**
Ja, das System ist für Endnutzer:innen kostenfrei zugänglich.

**Welche Sprachen werden unterstützt?**
Derzeit hauptsächlich Deutsch. Einfache Sprachmodi (Einfach, Leicht)
reduzieren die sprachliche Hürde zusätzlich. Mehrsprachigkeit ist in
Planung.

**Wie aktuell sind die Informationen?**
Die Informationsdatenbank wird regelmäßig geprüft und aktualisiert. Das
System zeigt keine Informationen an, deren Quelle als veraltet markiert ist.

**Kann das System für Beratungssitzungen genutzt werden?**
Ja – Fachkräfte können das System als schnelles Nachschlagewerk während
einer Beratung einsetzen. Es eignet sich besonders für Fragen wie „Welche
Rechte hat mein:e Klient:in in dieser Situation?" oder „Welchen nächsten
Schritt sollte ich empfehlen?"

**Was passiert mit den eingegebenen Fragen?**
Fragen werden verarbeitet, um eine Antwort zu liefern. Sie werden nicht mit
persönlichen Profilen verknüpft. Fragen, die eine redaktionelle Überprüfung
auslösen, werden anonymisiert gespeichert, damit das System verbessert
werden kann.
