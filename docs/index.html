Systemfehler

Digital tools for radical access: countering the failure of the welfare state with open-source solidarity.

What is this?

Systemfehler is an open-source platform designed to help individuals access welfare resources, foster community support, and document institutional challenges.

Quick Start

  1. Clone the repo: git clone https://github.com/steffolino/systemfehler.git
  2. See README.md for setup instructions.

Tech Stack

  • Frontend: Vue.js, TypeScript, JavaScript, CSS
  • Backend: Python
