# Agent Notes: Systemfehler Light CI

Follow the light CI and use brand assets sparingly.

## Use These Assets

- `BrandIcon` for compact branding
- `BrandWordmark` for header or primary identity areas
- `BrandHeroGraphic` for the main hero area only

## Prefer

- calm, legible layouts
- clear hierarchy
- one strong brand moment per screen
- decorative graphics only in hero-like areas

## Avoid

- inserting graphics everywhere
- using images to duplicate ordinary content
- adding decorative elements to forms, tables, or dense lists
- making the UI feel busy, playful, or promotional

## Decision Rule

If the graphic does not improve orientation or trust, do not use it.
